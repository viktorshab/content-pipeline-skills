# Saveable Strategy for Carousels

Карусель должна иметь причину для сохранения или пересылки. Этот файл помогает выбрать архетип и практический payoff.

## Choose one archetype

### 1. Checklist

Когда использовать:

- есть процесс;
- есть самопроверка;
- есть повторяемое действие.

Финальный слайд: чеклист.

### 2. Myth vs reality

Когда использовать:

- аудитория верит в неправильную причину;
- есть сильный wrong enemy;
- нужен shift in belief.

Финальный слайд: новая формула мышления.

### 3. Decision tree

Когда использовать:

- человеку нужно выбрать действие;
- есть разные сценарии;
- ошибки происходят из-за неверного выбора.

Финальный слайд: дерево решений.

### 4. Before / after / bridge

Когда использовать:

- есть трансформация;
- есть кейс;
- есть путь от состояния A к B.

Финальный слайд: bridge steps.

### 5. Teardown

Когда использовать:

- есть пример ошибки;
- можно разобрать скрин, фразу, упражнение, письмо, лендинг, профиль;
- аудитория любит разборы.

Финальный слайд: что исправить первым.

### 6. Swipe lesson

Когда использовать:

- один микро-урок;
- простой механизм;
- тема хорошо дробится на шаги.

Финальный слайд: summary formula.

### 7. Proof carousel

Когда использовать:

- есть данные, кейс, отзыв, before/after;
- нужно доверие;
- аудитория скептична.

Финальный слайд: what this proves and what it does not prove.

### 8. Script / template

Когда использовать:

- аудитории нужна готовая фраза;
- есть переписка, звонок, продажа, переговоры, self-check;
- можно дать шаблон.

Финальный слайд: ready-to-use script.

## Saveability check

Перед финалом ответь:

- Что человек сохранит?
- Что он сможет применить через 5 минут?
- Какой слайд хочется заскринить?
- Есть ли практический артефакт?
- Последний слайд дает пользу без перехода в профиль?

## Slide roles

| Role | Job |
|---|---|
| Cover | Stop scroll through tension |
| Recognition | Reader sees self |
| Reframe | Old belief breaks |
| Mechanism | New explanation |
| Proof/example | Trust |
| Practical slide | Save/screenshot |
| Final | Next step or artifact |

## CTA by archetype

- Checklist: "Сохраните и пройдите перед...".
- Myth vs reality: "Отправьте тому, кто все еще думает, что...".
- Decision tree: "Сохраните, чтобы выбрать следующий шаг".
- Teardown: "Хотите разбор своего случая, напишите...".
- Script/template: "Скопируйте этот шаблон и адаптируйте под себя".

CTA должен соответствовать продукту и температуре аудитории.
