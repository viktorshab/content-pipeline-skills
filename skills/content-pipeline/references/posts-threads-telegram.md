# Совместимый указатель: пост, тред, Telegram

Этот файл оставлен для обратной совместимости со старыми промтами.

Новая основная инструкция находится здесь:

```text
skills/content-pipeline/references/platform-playbooks-2026.md
```

Используй оттуда разделы:

- Instagram feed caption;
- X post;
- Threads;
- Telegram channel;
- LinkedIn expert post, если нужен B2B формат.

## Быстрое правило

Один источник не копируется между площадками 1-в-1.

- Instagram post: одна мысль, первая строка до раскрытия, save/share angle.
- X: sharp observation, single post или thread только если цепочка нужна.
- Threads: разговорное мнение, меньше механической нумерации.
- Telegram: пост для своих, больше контекста, мягче хук, живой голос.

Перед финалом прогоняй `quality-gates.md`.
