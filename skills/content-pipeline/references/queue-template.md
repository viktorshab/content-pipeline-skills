# Шаблон папки-очереди (раскладка пакета по датам)

Сборщик складывает готовый пакет в папку-очередь: каждая единица контента подписана площадкой, датой выхода и статусом. На выходе — не куча файлов, а понятный план недели. Образец ниже — заполняется под конкретный прогон.

## Правила раскладки

- **Одна единица = один текст + его картинка** (карусель = слайды, у неё отдельной картинки нет).
- **Статус всех новых единиц — «черновик».** На «утверждено» переводит человек руками, когда проверит. Сборщик сам в «утверждено» не ставит.
- **Дата** — день публикации. Не лепи всё на один день: разнеси по неделе так, чтобы форматы чередовались (тяжёлый разбор — лёгкий интерактив — личный пост).
- **Публикация** — из этой очереди, по одному, с подтверждением (звено публикатора). Очередь сама наружу ничего не отправляет.

## Структура папки на диске

```
проект_<ниша>/
  00-источник/            ← разбор аудитории, материал-источник
  10-пакет-неделя/
    01-post.md
    02-thread.md
    03-carousel/          ← слайды от carousel-builder
    04-reels.md
    05-telegram.md
    06-email.md
    images/               ← картинки под форматы (4:5 / 9:16 / 16:9)
  queue.md                ← эта таблица, заполненная под прогон
```

Имена — человеческие, чтобы было видно, что где. Картинки — рядом с текстами, чтобы единица собиралась целиком.
