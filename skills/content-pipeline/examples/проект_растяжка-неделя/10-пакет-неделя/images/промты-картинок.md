# Картинки под форматы

> Карусель собирается через `carousel-builder`. Для остальных форматов ниже только prompts и asset notes. Перед генерацией проверить `skills/carousel-builder/references/06-engine-passport.md`.

## Asset plan

| Format | Ratio | Need | Status |
|---|---|---|---|
| Instagram post | 4:5 | cover with clean text zone | prompt draft |
| X / Threads | 16:9 | optional visual metaphor | prompt draft |
| Reels / TikTok / Shorts | 9:16 | cover or first frame | prompt draft |
| Telegram | optional 16:9 | can publish text-only | optional |

## Prompt: Instagram cover

```text
Create a warm expert wellness social media cover, vertical 4:5. Scene: calm stretching studio, soft daylight, person preparing for a gentle forward fold, no pain expression. Leave clean text zone in upper third. Mood: supportive, not medical, not dramatic. Avoid generic stock photo, overdone fitness aesthetic, extreme split pose, pain expression, medical claims, random text. Text zone reserved for: "Тело не деревянное".
```

## Prompt: X / Threads visual

```text
Create a simple visual metaphor in 16:9: a soft knot or lock shape next to a yoga mat, warm studio light, minimal composition, clean negative space. No text, no logos, no medical imagery.
```

## Prompt: Reels cover

```text
Create 9:16 reel cover, warm stretching studio, close-up confident calm face or anonymous body preparing for a forward fold. Leave safe text zone in center. No extreme flexibility, no pain, no text rendered if code-text mode is used.
```

## Notes

- If text accuracy matters, use code-text mode.
- If using face reference, apply identity-lock from `07-лицо-identity.md`.
- Do not state price or model name unless engine passport is current.
