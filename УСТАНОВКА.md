# Установка и запуск — по шагам

Здесь: как поставить скиллы к вашему код-агенту и готовые промты под каждый шаг. Принцип урока — **руками вы почти ничего не делаете, командуете агенту словами.**

---

## Часть 1. Поставить скиллы (один раз)

### Шаг 1. Скачать
Два способа, любой:
- На странице репозитория — зелёная кнопка **Code → Download ZIP**, распакуйте.
- Или в терминале: `git clone https://github.com/viktorshab/content-pipeline-skills.git`

### Шаг 2. Положить скиллы к агенту
Скопируйте папки из `skills/` в папку скиллов вашего код-агента.

**Claude Code** читает скиллы из двух мест:
- для одного проекта — `<ваш-проект>/.claude/skills/`
- для всех проектов сразу — `C:\Users\<вы>\.claude\skills\` (Windows) или `~/.claude/skills/` (Mac/Linux)

Должно получиться так:
```
.claude/skills/content-pipeline/SKILL.md
.claude/skills/carousel-builder/SKILL.md
.claude/skills/reels-scriptwriter/SKILL.md
.claude/skills/email-sequence-writer/SKILL.md
```

Проще не копировать руками, а поручить агенту:
```
Скопируй папки скиллов из [путь к распакованной папке skills] в .claude/skills/ моего проекта.
Покажи, что в итоге лежит в .claude/skills/.
```

> Если работаете в Codex — положите эти же папки в каталог скиллов вашего агента (механизм может отличаться, уточните в момент работы).

### Шаг 3. Проверить, что подхватилось
```
Покажи, какие скиллы тебе сейчас доступны.
```
В списке должны быть: **content-pipeline, carousel-builder, reels-scriptwriter, email-sequence-writer**. Если какого-то нет — проверьте, что его папка лежит в `.claude/skills/` и внутри есть `SKILL.md`.

## Часть 2. Собрать неделю контента (готовые промты под каждый шаг)

Запускайте в папке вашего проекта, где лежит разбор аудитории (боли, портрет клиента, язык клиентов) — это топливо. Если разбора нет, источником может быть статья, запись эфира или ссылка на видео.

### Шаг 1. Источник → тема недели
```
Возьми как источник [папку моего проекта с разбором аудитории / файл / ссылку на видео].
Если это видео или запись — сначала расшифруй в текст.
На основе болей моей аудитории предложи 3–5 тем недели, под которые есть что сказать.
Покажи их списком — я выберу одну, и только потом пойдём собирать форматы.
```
*Нормально, если:* список тем из ваших реальных болей, а не из воздуха.

### Шаг 2. Запустить сборщика и проверить, что он нашёл кубики
```
Запусти сборщика контента (content-pipeline) на выбранной теме.
Сначала покажи списком, какие готовые скиллы-ассистенты под форматы ты нашёл, — я подтвержу.
Те, что есть, — вызывай, а не пиши заново.
```
*Нормально, если:* агент назвал carousel-builder, reels-scriptwriter, email-sequence-writer — то, что вы установили.

### Шаг 3. План пакета (стоп-гейт)
Агент показывает короткий план: какой формат о чём и под какую площадку. **Здесь правьте направление — это бесплатно.** Полные тексты он разворачивает только после вашего «да».

### Шаг 4. Развернуть форматы
```
Разверни пакет: пост, тред, карусель, сценарий Reels, сообщение в Telegram и письмо —
каждый под свою площадку, не один текст на всех. Голос бренда держи из разбора аудитории.
Карусель собирай через carousel-builder, Reels — через reels-scriptwriter,
письмо — через email-sequence-writer. Reels: переопредели нишу под мой проект (не оставляй дефолтную).
```
*Нормально, если:* у каждого формата свой ритм под площадку, везде ваш голос.

### Шаг 5. Картинки в нужных размерах
```
Дорисуй картинку под каждый формат, сразу в правильном размере под площадку:
лента и карусель — 4:5, сторис и Reels — 9:16, тред и пост в X — 16:9.
Карусель пропусти — она уже со слайдами.
Сначала покажи, сколько всего картинок выйдет и примерно почём, — я подтвержу, потом генерируй.
Если в кадре нужно моё лицо или товар — держи их по моему фото-референсу.
```
*Нормально, если:* к каждому тексту своя картинка в правильном размере. Картинок на неделю много — поэтому сначала объём и цена.

### Шаг 6. Разложить по датам
```
Разложи готовый пакет в папку-очередь: каждую единицу (текст + её картинка) подпиши
площадкой и датой публикации. Статус у всех — «черновик», на «утверждено» я переведу сам.
Покажи итог таблицей: дата, площадка, формат, тема.
```
*Нормально, если:* на выходе папка-план недели, а не куча файлов.

## Что остаётся руками

- **Фото вашего лица/товара** — если хотите их в кадре (для карусели, поста, Reels).
- **Рендер картинок** — кнопку «сгенерировать» нажимаете вы, вашим image-инструментом.
- **Публикация** — по одному посту, с вашим подтверждением. Массовый автопостинг «во все соцсети сам» не делаем — это путь к бану.
- **Финальная вычитка** перед публикацией — ваш голос, ваша ответственность.

Это и есть неделя контента из одного материала. Дальше можно тем же агентом собрать план на месяц.
